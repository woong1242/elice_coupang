# Coupang 홈페이지 Testcase작성 실습
## 목표
**Selenium** 과 **pytest**를 이용해 쿠팡 홈페이지 기능을 테스트 해 본다.

## Project setup
- pytest 설치
```python
pip install pytest
```
- Selenium 설치
```python
pip install Selenium
```

## 필수 사항
- 주어진 6개 testcase에 대해 test 코드를 작성 한다.
1. 쿠팡 상품 검색
2. 쿠팡 장바구니
3. 쿠팡 상품 옵션
4. 쿠팡 검색 필터
5. 쿠팡 판매 특가
6. 쿠팡 최근 상품

- 각 단계 성공 여부를 Screenshot 으로 저장한다.
